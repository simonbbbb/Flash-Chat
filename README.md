# Flash-Chat

## Alkalmazás bemutatása

Az alkalmazás felhasználók közötti valós idejű üzenetváltást tesz lehetővé.

## Alkalmazás működése

Az alkalmazás felhasználók közötti valós idejű üzenetváltást tesz lehetővé. A felhasználói adatokat egy Firebase adatbázi kezeli. 

## Forrás

Az alkalmazás létrehozása során igénybevett segédanyag:

London App Brewery - The Complete iOS App Developement Bootcamp kurzusán található Skeleton fájl mely tartalmazta a megjegyzésben található útmutatókat és grafikai elemeket. A skeleton fájl elérhető: https://github.com/londonappbrewery/Flash-Chat-iOS12
